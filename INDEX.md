## 🚀 How to Run

### Option 1: Interactive Launcher (Recommended)
```bash
python launch.py
```
Choose from menu:
- 1: Full mode (backend + UI)
- 2: Demo mode (UI only)
- 3: View docs
- 4: Exit

### Option 2: Manual Full Setup
```bash
cd backend
pip install -r requirements.txt
python app.py
```
Then open: http://localhost:5000

### Option 3: Quick Demo
Open `frontend/index.html` in browser (no setup needed!)
